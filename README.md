# css-default

[![Build Status](https://api.travis-ci.org/vigour-io/css-default.svg)](https://travis-ci.org/vigour-io/css-default)

Minimal CSS reset for Vigour apps

## Getting started

### Install it:
```shell
npm i @vigour-io/css-default --save
```

### Require it:
```js
require('@vigour-io/css-default')
```
